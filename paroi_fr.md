# paroi

**Domaine:** ARCH  
**Information linguistique:** s. m.  
**Pays:** FR  

### Définition
Cloison de maçonnerie qui sépare les pièces à l'intérieur d'une maison, d'un bâtiment.
(1)

### Contexte
*A l’intérieur la paroi est décorée de trois raies blanches à distance régulière l’une de l’autre. Pied très bas, tronconique ; la surface interne convexe présente de menus éclats de pierre ; surface externe lisse.
(3)*

### Note
Généralement le terme indique la surface intérieure d’un mur de maison, d’un bassin, d’un récipient.
(2)
