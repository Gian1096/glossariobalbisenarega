# timpano

**Dominio:** ARCH  
**Informazione grammaticale:**   
**Etichetta paese:** IT  

### Definizione
[Struttura architettonica corrispondente alla] superficie racchiusa entro la cornice del frontone, concepita originariamente come una membrana avente scopo di chiusura, senza funzione portante.
(1)

### Note
Gli edifici provvisti di tale elemento si dicono timpanati.
