# palier d’escalier

**Domaine:** ARCH  
**Information linguistique:** s. m.  
**Pays:** FR  

### Définition
Plateforme entre deux volées d’un escalier.
(1)

### Contexte
*Chaque palier est orné d’une colonne engagée où le chapiteau porte les caractéristiques de décorations grecolatines de l’antiquité qui seront repris à l’époque de la Renaissance.
(3)*

### Note
Il s’agit d’un espace ou d’une plate-forme servant de repos dans un escalier, dans un perron, dans une rampe douce, ou dans les gradins d’un théâtre.
On peut parler de palier de communication, quand on veut identifier celui qui est entre des appartements de plain-pied, et qui leur est commun.
[Adaptée de (2)]
