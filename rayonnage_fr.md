# rayonnage

**Domaine:** ARCH  
**Information linguistique:** s. m.  
**Pays:** FR  

### Définition
Ensemble de planches fixées horizontalement pour constituer une bibliothèque, une vitrine, un meuble de rangement, etc.
(1)

### Contexte
*Un rayonnage chargé de caisses laisse difficilement circuler l’air, ce qui influe sur le climat. C’est pourquoi l’anticipation de l’implantation des étagères de rangement est une étape indispensable pour obtenir un climat satisfaisant dans les réserves.
(2)*
